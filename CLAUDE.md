# CLAUDE.md

This project's guidance lives in [AGENTS.md](./AGENTS.md) — please read it.

In short: agent-agnostic genealogy skills for reading, building, analysing and
visualising GEDCOM (`.ged`) family trees. Pure Python 3 stdlib via `bash`, no
dependencies. The skills are in `skills/`; try them against `examples/demo.ged`.
Never commit real family `.ged` files (only `examples/demo.ged` is tracked).
